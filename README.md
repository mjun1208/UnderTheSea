## 게임 설명 : 
- DirectX9으로 만든 게임입니다.
- 포지션 값과 맵 사이즈를 20배줄이는 형식으로 미니맵을 구현했습니다.
- 미니맵에서 픽셀충돌을 사용하여 충돌처리를 구현했습니다.
- 적과 총알은 AABB Rect 충돌입니다.
